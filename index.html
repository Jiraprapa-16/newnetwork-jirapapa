ประเภทเครื่องแม่ข่าย

ประเภทเครื่องแม่ข่ายในระบบเครือข่ายคอมพิวเตอร์

Web Server (เครื่องแม่ข่ายเว็บไซต์)

เครื่องแม่ข่ายประเภทนี้ทำหน้าที่ให้บริการเว็บไซต์ โดยจะเก็บข้อมูลของเว็บไซต์ (เช่น HTML, CSS, JavaScript, ภาพ) และตอบสนองคำขอของผู้ใช้ผ่านโปรโตคอล HTTP หรือ HTTPS

ซอฟต์แวร์ที่ใช้: Apache HTTP Server, NGINX, Microsoft IIS

Database Server (เครื่องแม่ข่ายฐานข้อมูล)

เครื่องแม่ข่ายประเภทนี้ทำหน้าที่ในการจัดการและให้บริการข้อมูลในรูปแบบฐานข้อมูล เช่น MySQL, PostgreSQL, Microsoft SQL Server และ Oracle Database

ซอฟต์แวร์ที่ใช้: MySQL, PostgreSQL, Microsoft SQL Server

File Server (เครื่องแม่ข่ายไฟล์)

เครื่องแม่ข่ายประเภทนี้ทำหน้าที่ในการเก็บและแชร์ไฟล์ให้กับผู้ใช้ในเครือข่าย เช่น การแชร์ไฟล์ในองค์กร

ซอฟต์แวร์ที่ใช้: Samba, Windows Server File Services, NFS
